# 🤖 AI Career Path Generator

An intelligent web application that generates personalized career roadmaps using AI based on user interests, skills, and goals.

This project helps students and aspiring professionals discover structured learning paths tailored to their career ambitions.

---

## 🚀 Features

* 🎯 Input your:

  * Current Education Level
  * Skills
  * Interests
  * Target Career Role
* 🤖 AI-generated personalized career roadmap
* 📚 Suggested technologies and tools to learn
* 🛠 Step-by-step learning progression
* 🧠 Skill gap analysis
* ⚡ Instant response using AI API integration
* 💻 Clean and responsive UI

---

## 🛠 Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### AI Integration

* Groq API / LLM API Integration

---

## 🎯 Project Objective

The goal of this project is to:

* Help students make informed career decisions
* Provide structured guidance instead of random learning
* Apply AI in real-world practical use case
* Strengthen full-stack development skills

---

## 🧠 How It Works

1. User enters career preferences and background.
2. Backend sends structured prompt to AI model.
3. AI generates a customized roadmap.
4. The roadmap is displayed in a readable format on the UI.

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/yourusername/AI-Career-Path-Generator.git

2. Navigate to Backend folder:
   cd Backend

3. Install dependencies:
   npm install

4. Create a `.env` file and add:
   GROQ_API_KEY=your_api_key_here

5. Start server:
   npm start

6. Open frontend in browser.
